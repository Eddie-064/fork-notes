# EC2 的网络限制

Linux 系统中的内核参数限制了机器的网络性能，对于边缘网关（入口或出口网关）场景通常都需要调整这些参数。
但在 EC2 中，即使你调高了这些 Linux 内核参数，也不一定能达到你想要的网络性能，因为 EC2 本身也有硬性的网络限制。

所以可能遇到的情况是，你机器监控显示它网络性能还有很大冗余，但实际已经开始出问题了，这时候就需要考虑 EC2 的网络限制了。
