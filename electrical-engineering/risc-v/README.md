# RISC-V 系列

RISC-V 是纯开源的指令集，在最近几年声名鹊起，被中国等许多国家视为掌握国家芯片安全的最佳途径，正在 MCU、嵌入式等领域蓬勃发展中。

我目前拥有的 RISC-V 开发板有：

- ESP32-C3: 乐鑫家的 RISC-V 芯片，便宜好用
- Lichee Pi 4A: 用的阿里平头哥曳影 1520 芯片，理论性能比 Raspberry Pi 4B 还要强一截。
  - 预定了两个，已到手一个内测评估板，还没开始玩。另一个券打算出正式板再用。
